Project Description 
EU Development Analysis using Hierarchical Clustering
This project analyzes the economic and social development patterns of European Union member states using hierarchical clustering techniques in Python. The objective of the analysis is to identify groups of countries with similar development characteristics based on multiple economic, social, and sustainability indicators.
The dataset used in this project comes from the World Bank World Development Indicators (2022) and includes variables such as GDP per capita, life expectancy, education participation, energy access, research and development spending, and energy efficiency.
The analysis applies unsupervised learning techniques to explore the structure of the data and identify clusters of countries with similar characteristics. Hierarchical clustering using the Ward method and Euclidean distance is implemented to group the countries based on standardized indicators.
The results highlight structural differences between EU countries, revealing distinct clusters that broadly reflect development patterns between Eastern and Western Europe. The project demonstrates how data analysis and clustering techniques can be used to uncover meaningful patterns in economic and social data.

Objective
